# Netflix_Movie-TVshow-Analysis
This repository contains an analysis of Netflix Movies and TV Shows using T-SQL in MS SQL Server. The project involves in-depth exploration of the Netflix dataset to uncover insights and trends in streaming content. Key features include:

Data Exploration: Querying the dataset to understand content distribution across genres, release years, and regions.
Insights Generation: Identifying patterns such as the most popular genres and trends in content production.
Advanced SQL Techniques: Use of CTEs, subqueries, aggregate functions, and window functions for data analysis.
Recommendations: Data-driven insights to assist in content strategy and user engagement.
